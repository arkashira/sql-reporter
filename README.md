# SQL Reporter

A simple SQL query execution engine.

## Usage

1. Create a dataset dictionary with table names as keys and lists of dictionaries as values.
2. Create an instance of the `SQLReporter` class, passing the dataset dictionary to the constructor.
3. Execute a query using the `execute_query` method, passing the query string as an argument.
4. Format the result using the `format_result` method, passing the query result as an argument.
